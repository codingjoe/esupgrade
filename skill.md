# Agent Code Writing Skill

Use this skill when writing or updating code in this repository.

## Requirements

1. Ensure transformation safety first.
2. Follow the [naming guidelines](https://raw.githubusercontent.com/codingjoe/naming-things/refs/heads/main/README.md).
3. Read and apply [CONTRIBUTING.md](./CONTRIBUTING.md) before changing code.
4. Keep code fully tested with 100% coverage and remove unreachable branches.
5. Run pre-commit hooks before finalizing changes.

## Naming Guidelines Command

```console
curl -sSL https://raw.githubusercontent.com/codingjoe/naming-things/refs/heads/main/README.md | head -n 500
```
